EXAMPLE DESCRIPTION
==============================================================================
    This directory contains a simple command-line, menu-driven program for
    entering and viewing records in a technical information database. In
    addition to the source code for the program, a pre-loaded TIMS database
    and import script are included. The import script can be used with
    the dbimp utility (dbimp tims.imp). The TIMS program is referred to
    extensively in the RDM User's Guide.

==============================================================================
FILE LIST DESCRIPTION
==============================================================================
Source Files
    borrow.c      - Publication borrowing functions
    delinfo.c     - Delete publication info function
    entinfo.c     - Enter publication info function
    findpubs.c    - Find publications by various methods
    listauth.c    - Produce author list
    listkeys.c    - Produce key_word list
    timsExample_main.c  - TIMS main program
    tims_dbd.c    - Source generated by ddlproc containing database structure

Header and Database Files
    tims.h        - Header generated by ddlproc containing prototypes
    tims_dbd.h    - Header generated by ddlproc containing database structure
    tims.imp      - Import script. Can be used by dbimp.
    tims.ddl      - TIMS schema
    tims.dbd      - TIMS Data Dictionary file
    tims.d01      - TIMS Database.  Data file.
    tims.d02      - TIMS Database.  Data file.
    tims.k01      - TIMS Database.  Key file.
    tims.k02      - TIMS Database.  Key file.
    tims.v01      - TIMS Database.  Vardata file.
    tims.v02      - TIMS Database.  Vardata file.

Data Files
    author.txt    - Used in tims.imp. author records.
    info.txt      - Used in tims.imp. info records.
    intersct.txt  - Used in tims.imp. intersct records.
    key_word.txt  - Used in tims.imp. key_word records.
    text.txt      - Used in tims.imp. text records.
