Summary Listing of Utility API d_ Functions

The following table lists alphabetically all the Utility API d_ functions, along with brief descriptions. All functions have int32_t as their return types.

These functions require the header file "rdm.h" included in source files. To link these functions, the library name 'util' must be included (see Library Naming Conventions for library naming conventions).

Function Description
d_dbcheck Check database consistency (see dbcheck)
d_dbcluster Database set optimization (see dbcluster)
d_dbdefrag Defragment database files (see dbdefrag)
d_dbexpcsv Database ASCII export (see dbexp).
d_dbexpxml Database XML export (see dbexp).
d_dbrevise Defragment database files (see dbrev)
d_dbswap Database byte-swap (see dbswap)
d_dchain Delete chain sort API (see dchain)
d_keybuild Rebuild key files (see keybuild)
d_ddlp Database definition language processor API. (see ddlp)
