Summary Listing of Transaction Control and Locking Functions

The following table lists alphabetically all the transaction and locking functions, along with brief descriptions. All functions have int32_t as their return types.

Transaction Control

Transactional control by use of the following functions specified below:

Function Description
d_trbegin Start a transaction
d_trend Commit current transaction
d_trabort Abort current transaction
d_trprecommit Precommit current transaction
d_trrobegin Begin Read-Only transaction
d_trroend End Read-Only transaction
d_trmark Mark a transaction
d_trrollback Roll a transaction back to a mark
d_trdeletemark Remove a transaction mark
d_tractive Determine whether a transaction is active

Locking

Locking can be established using the following functions:

Function Description
d_reclock Lock files associated with record type
d_reclstat Get lock status of record type
d_recfree Free locks being held on record type
d_keylock Lock key file associated with key type
d_keylstat Get lock status of key type
d_keyfree Free locks being held on key type
d_setlock Lock files associated with set type
d_setlstat Get lock status of set type
d_setfree Free locks being held on set type
d_lock Lock group of record, key and/or set types
d_freeall Free al locks being held
d_timeout Set the lock request timeout value
