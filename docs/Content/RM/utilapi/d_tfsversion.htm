d_tfsversion

Query TFS library version and configuration information

Prototype

int32_t d_tfsversion(
    uint16_t     *major, 
    uint16_t     *minor, 
    TFS_LIB_TYPE *type);

Parameters

major (output) The major version number of RDM.
minor (output) The minor version (revision) number of RDM.
type (output) The underlying TFS type: RPC, Direct-link or Standalone.

Description

This function asks the TFS library which configuration it is. The type parameter receives an enumerated value back. There are three possible values:

  • TFS_LIB_TYPE_RPC - [DEFAULT] The TFS is running as a separate server and the runtime accesses the TFS through Remote Procedure Calls.
  • TFS_LIB_TYPE_TFS - The TFS has been linked directly into this program.
  • TFS_LIB_TYPE_STANDALONE - The TFS supports only a single user and single thread. Meant for specialized environments.

In addition, the major and minor version of RDM are returned.
