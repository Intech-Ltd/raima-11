d_tfsIterateConnectedUsers

Query names of all users currently connected to a TFS.

Prototype

int32_t d_tfsIterateConnectedUsers(
    TFS_HANDLE               hTFS,
    const char              *hostname,
    uint16_t                 port,
    TFS_ITERATE_USERS_FCN    handlerFcn)

int32_t d_tfsIterateConnectedUsersW(
    TFS_HANDLE               hTFS,
    const wchar_t           *hostname,
    uint16_t                 port,
    TFS_ITERATE_USERS_W_FCN  handlerFcn)

Parameters

hTFS (input) Handle returned from a successful d_tfsinit call.
hostname (input) Hostname of the TFS to query for user names.
port (input) Port number of the TFS to query for user names.
handlerFcn (input) Callback function, called once for each user name connected to the TFS.

Description

This function queries the TFS to return all user names for all connections currently logged in to the TFS.

The program calling this function will call d_tfsIterateConnectedUsers once. The handlerFcn will be called multiple times by the runtime library, once for each TFS connection.

The handlerFcn will return S_OKAY unless the program no longer needs to receive more user names. Returning any error code will stop the runtime from calling handlerFcn.

User names that begin with "RDM." are internal connections and should never be disconnected.

Only one instance of d_tfsIterateConnectedUsers from any application can be executing at a given point in time for all calls to this function for a given TFS.

TFS_ITERATE_USERS_FCN Prototype

typedef int32_t (EXTERNAL_FCN *TFS_ITERATE_USERS_FCN)(const char *userName);

or

typedef int32_t (EXTERNAL_FCN *TFS_ITERATE_USERS_W_FCN)(const wchar_t *userName);

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-200 S_TX_ERROR generic tx_ error
-202 S_TX_NETREAD failure to read data from network
-203 S_TX_NETWRITE failure to write data to network
-205 S_TX_DUPUSER same dbuserid already in use
-214 S_TX_CONNECT failed to connect to TFS
-215 S_TX_HOSTNAME TFS host name not found
-904 S_NOMEMORY out of memory
-924 S_INVNULL invalid NULL parameter

Example

int32_t EXTERNAL_FCN UserName_handler(
    const char     *username)
{
    TFS_HANDLE    hTFS = NULL;

    /* This function is called once for each connected user to a TFS.
       Return S_OKAY to continue receiving user names.
       Return any other error to stop receiving user names.
       Ignore internal RDM connections.
    */

    if (strncmp(username, "RDM.", 5) == 0)
        return S_OKAY;    /* ignore internal connections */

    printf("Username: %s\n", username);

    if (strcmp(username, "my_user") == 0)
    {
        if (d_tfsdisconnect(hTFS, "localhost", 21553, username) == S_OKAY)
        {
            printf("Disconnected user %s\n", username);
            return S_EOS;  /* Return error to stop getting new user names */
        }
        else
            printf("Error disconnecting user\n");
    }

    return S_OKAY;
}
