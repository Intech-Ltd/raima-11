d_dbswap

Database byte-swap utility API.

Prototype

int32_t d_dbswap(
    const char          *dbname,
    const DBSWAP_PARAMS *params);

int32_t d_dbswapW(
    const char           *dbname,
    const DBSWAP_PARAMSW *params);

Parameters

dbname (input) The name of the database that will be byte-swapped. The full file path must be specified.
params (input) A pointer to a DBSWAP_PARAMS structure.

Description

The dbswap utility performs byte-swap operations on a database allowing it to be used on a platform utilizing an integer endian format different than the platform the database was created on. See the dbswap utility page for a description of its operation.

DBSWAP_PARAMS structure
Member Type Description
quiet DB_BOOLEAN No output except for errors.
verbose DB_BOOLEAN Show detailed output of the operations performed on the database
overwrite int16_t Flag to specify if you want to overwrite the existing database files
dbd_only int16_t Flag to tell the utility to swap the dbd file, but not the data, key, and varchar files.
nonnative int16_t Flag that specifies the database is a byte-order format not native to the system the utility is running on
stdout_file const char * This specifies the name of the file to which output data will be written. If set to PSP_STDOUT, stdout is used. If set to NULL or to an empty string, no output will be generated.

Currency Changes

The utility does not use the RDM runtime and therefore currency is not affected.

Locking Requirements

This utility does not use the RDM runtime and therefore locking in not required or respected. You should make sure that no threads/process are utilizing the database image before starting the byte-swapping process.

Required Headers

#include "dutilproto.h"

Libraries

Library Name Description
rdmutil11 RDM Utility Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-4 S_INVDB cannot open dictionary
-21 S_INCOMPAT incompatible dictionary file
-904 S_NOMEMORY out of memory
-914 S_BADREAD error reading from a data or key file
-919 S_BADWRITE error writing to a data or key file

See Also

dbswap utility

Example

/* RDM Database Byte-swap Utility API Example ******************************
*/
int32_t MAIN(int32_t argc, const char *argv)
{
    /* Because dbswap does not use the runtime, you must specify the full dbname path */
    char         *dbname = "C:\\RDM\\databases.win32\\tims\\tims";
    DBSWAP_PARAMS params;
    int32_t       status = S_OKAY;

    psp_init();
    memset(&params, 0, sizeof(DBSWAP_PARAMS));

    params.verbose   = 1;
    params.overwrite = 0;
    params.dbd_only  = 0;
    params.nonnative = 0;
    params.stdout_file = PSP_STDOUT;

    status = d_dbswap(dbname, &params);

    psp_term();
    return(0);
}
