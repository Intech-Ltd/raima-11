d_dbexpxml

Database XML export utility API.

Prototype

int32_t d_dbexpxml(
    const DBEXPXML_PARAMS *params,
    DB_TASK               *task,
    int32_t                dbn);

int32_t d_dbexpxmlW(
    const DBEXPXML_PARAMSW *params,
    DB_TASK                *task,
    int32_t                 dbn);

Parameters

params (input) A pointer to a DBEXPXML_PARAMS structure.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This API exports data from the database to an XML file. By exporting data into XML format, consumers of the data can be RDM (e.g. for database restructuring or archiving), RDM Server, RDM Mobile, or other XML consumers, including other database products such as ADO.NET. The XML content created by the d_dbexpxml API will allow reconstruction of sets (in the case of RDM as the consumer), or interpretation by any normal XML consumer.

DBEXPXML_PARAMS structure
Member Type Description
quiet DB_BOOLEAN No output except for errors.
verbose DB_BOOLEAN The -v option. Show detailed output of the operations performed on the database
decimal DB_BOOLEAN Not valid for XML export.
extended DB_BOOLEAN The -e option. Print extended ASCII characters (default is to use octal).
n_recs uint16_t The number of record names listed in the recnames parameter. This should be 0 if you want to export all records in the database.
recnames const char ** An array of record names listing the specific records that will be exported. If this list is empty all records in the database will be exported.
out_file const char * Name of the output file for the export. If this is NULL the default with dbname.xml
schema_type int32_t

The -xsd or -dtd options. Specify the type of XML schema to export with the database data. The allowed values are:

  • DB_EXP_XML_NONE – no schema
  • DB_EXP_XML_SCHEMA – XML schema
  • DB_EXP_XML_DTD – XML DTD
stdout_file const char * This specifies the name of the file to which output data will be written. If set to PSP_STDOUT, stdout is used. If set to NULL or to an empty string, no output will be generated.
printfData void * Deprecated. Set to NULL.
openfiles uint8_t Deprecated. Set to 0.

Currency Changes

This utility will invalidate currency.

Locking Requirements

This utility requires exclusive read to the database. If you open the database in shared 's' mode you must get read ('r') locks on all database files.

Example

/* RDM Database XML Export Utility API Example ******************************
*/
int32_t MAIN(int32_t argc, const char *const *argv)
{
    int32_t          status = S_OKAY;
    DB_TASK         *task;
    DBEXPXML_PARAMS  params;

    memset(&params, 0, sizeof(DBEXPXML_PARAMS));

    /* just set the non-zero parameters */
    params.verbose     = TRUE;
    params.schema_type = DB_EXP_XML_SCHEMA;
    params.stdout_file = PSP_STDOUT;

    if ((status = d_opentask(&task)) == S_OKAY) {
        if ((status = d_open("tims", "x", task)) == S_OKAY) {
            status = d_dbexpxml(&params, task, CURR_DB);
            d_close(task);
        }
        d_closetask(task);
    }

    return(0);
}
