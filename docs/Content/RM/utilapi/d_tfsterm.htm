d_tfsterm

Clean up resources allocated by TFS initialization

Prototype

int32_t d_tfsterm(
    TFS_HANDLE hTFS);

Parameters

hTFS (input) Handle returned from a successful d_tfsinit call.

Description

Any successful call to d_tfsinit requires a corresponding call to d_tfsterm.. Any calls to d_tfsrun must be followed by d_tfsstop before calling this function. The final call to d_tfsterm should occur after the final call to d_closetask.
