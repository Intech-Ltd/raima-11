d_tfsxapiparams

Provide TFS with parameters to start up as a HTTP server

Prototype

int32_t d_tfsxapiparams(
    TFS_HANDLE       hTFS,
    TFS_XAPI_PARAMS  *pParams);

Parameters

hTFS (input) TFS handle.
pParams (input) TFS_XAPI_PARAMS

Description

This function initializes the TFS functionality so that it can operate as a HTTP server. The TFS HTTP server can serve simple HTML files as well as handle the database functionality requested by RDM runtimes in other processes. The names and addresses of additional functions to be made available through HTTP may also be provided, allowing extension functions to be invoked from HTTP clients.

The function d_tfsversion will identify which configuration the application is actually running within at run time.

The Direct-Link configuration allows the application program to start and stop the listening and connection threads for the HTTP server. See functions d_tfsrun and d_tfsstop.

TFS_XAPI_FCNMAP structure definition
Element Declaration Description
name const char * Function name for reference in a URL.
fcn TFS_XAPI_FCN * Address of XAPI function to call.

 

TFS_XAPI_PARAMS structure definition
Element Declaration Description
xapifcnmap const TFS_XAPI_FCNMAP * Function map for HTTP XAPI functions.
fcnmapsize uint16_t Number of entries in xapifcnmap.
subdocroot const char * The directory under the "httpdocs" directory for HTML file location.

Example

int32_t navigate_handler(
    RHS_HANDLE      hRHS,
    PSP_OUTPUTBUFF  outbuff,
    const char     *args,                /* URL decoded args */
    const char     *raw_args,            /* URL encoded args */
    const char     *content)
{
    /* 'args' is a string containing everything after the '?', for example, if
       the reference is 'http://localhost:21553/navigate.rdm?action=d_recfrst&' then
       args will be 'action=d_recfrst&'.

       The content may be URL encoded if it is received from a standard
       web browser. If so, it must be decoded, as shown below.
    */
    ...
    if (content)
        http_URLDecode(content, NULL);
 
    /* produce output for browser */
    http_printf(outbuff, "name=%s&index=%d&", name, index);

    return stat;
}

main()
{
    static const TFS_XAPI_FCNMAP timsfcnmap[] = {
        { "/navigate.rdm", (TFS_XAPI_FCN *)navigate_handler  },
    };

    TFS_XAPI_PARAMS xapi_params;
    TFS_PARAMS    tparams;
    TFS_HANDLE    hTFS;
    int32_t       rc;
    uint16_t      tfs_done = 0;
    ...
 
    memset(&tparams, 0, sizeof(TFS_PARAMS));

    xapi_params.xapifcnmap = timsfcnmap;
    xapi_params.fcnmapsize = MAX_TFSFCNS;
    xapi_params.subdocroot = "/tims";
    ...
 
    d_tfsversion(&major, &minor, &type);
    if (type != TFS_LIB_TYPE_TFS)
    {
        printf("Incorrect TFS library loaded: type = %d\n", type);
        d_closetask(task);
        return 1;
    }
    
    rc = d_tfsinit(docroot, &tparams, &hTFS);
    
    if (rc == S_OKAY)
    {
        d_tfsxapiparams(hTFS, &xapi_params);

        /* Waits while server code processes requests */
        d_tfsrun(hTFS, FALSE, &tfs_done);
        
        d_tfsterm(hTFS);
    }
    ...
}
