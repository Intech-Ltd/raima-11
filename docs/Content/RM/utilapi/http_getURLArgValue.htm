http_getURLArgValue

Retrieve the value associated with a key

Prototype

const char *http_getURLArgValue(
    const char    *key,
    const URLARGS *urlargs);

Parameters

key (input) A string identifying the key associated with the requested value.
urlargs (input) Pointer to an allocated structure created by http_initURLArgs.

Description

This function searches through the key/value pairs represented in urlargs (must have been created by http_initURLArgs) for the key. If found, a pointer to the value associated with key is returned. If the key is not found, NULL is returned.

See the http_initURLArgs for more description and an example.

Required Headers

#include "rdmhttp.h"
