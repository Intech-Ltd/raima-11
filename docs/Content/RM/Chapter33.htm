Summary Listing of RDM Functions

The following table lists alphabetically all the RDM functions. All functions have int32_t as their return types.

These functions require the header file "rdm.h" included in source files. To link these functions, the library name 'rdm' must be included (see Library Naming Conventions for library naming conventions).

d_blobsize Retrieve the size of the BLOB data

d_blobtruncate Truncate a BLOB
d_blobwrite Write a BLOB

d_crwrite Write to field in current record
d_csmget Get database address of current member
d_csmread Read data from field of current member
d_csmset Set database address of current member
d_csmwrite Write to field in current member
d_csoget Get database address of current owner
d_csoread Read data from field of current owner
d_csoset Set database address of current owner
