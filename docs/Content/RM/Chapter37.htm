Summary Listing of Database Control Functions

Pre-Database Open Functions

These functions must be called prior to opening the database.

Pre-Database Open Functions
Function Description
d_dbuserid Set database user identifier.
d_dbver Obtain runtime library version information.
d_open Open a (set of) database(s).
d_open_ptr Open a database using a dbd array.
d_opentask Create a new task context.
d_setpages Set the number of pages in the cache.
d_dbini Set path to the RDM initialization file.

Post-Database Open Functions

These functions must be called after opening the database.

Function Description
d_close Close database.
d_closetask Close a task context.
d_destroy Destroy (delete) all data and key files in the database.
d_iclose Incrementally close a database.
d_initfile Initialize a database file.
d_initialize Initialize all database files.
d_iopen Incrementally open a (set of) database(s).
d_off_opt Turn off specified runtime options.
d_on_opt Turn on specified runtime options.
