Summary Listing of BLOB Functions

The following table lists alphabetically all the BLOB functions, along with brief descriptions. All functions have int32_t as their return types.

Function Name Description
d_blobdelete Delete BLOB data
d_blobread Read BLOB data
d_blobseek Seek to a given position in the BLOB data
d_blobsize Retrieve the size of the BLOB data
d_blobtell Retrieve the current byte offset position of BLOB data
d_blobtruncate Truncate a BLOB
d_blobwrite Write a BLOB
