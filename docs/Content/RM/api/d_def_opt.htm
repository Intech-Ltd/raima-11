d_def_opt

Reset RDM options to default values

Prototype

int32_t d_def_opt(
    uint32_t  optflag, 
    DB_TASK  *task);

Parameters

optflag (input) The option flag containing bit settings for the options to be reset.
task (input) A pointer to a DB_TASK structure.

Description

This function is used to reset any or all RDM runtime options to their default values.

The optflag argument contains a bit status integer that has a bit associated with each option. Header file rdm.h contains constant definitions for each option. The runtime option descriptions can be found in Runtime Options.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-48 S_DBCLOSE database not closed

Example

#include <stdio.h>
#include <rdm.h>

/* open in read only mode */
   d_on_opt(READONLY, task);
   d_open("local_db", "o", task);
   . . .
   d_close(task);
   
/* now reset for normal open */
   d_def_opt(READONLY, task);
   d_open("share_db", "s", task);
   . . .
