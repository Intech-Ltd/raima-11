d_decode_dba, d_decode_ldba

Decode database address

Prototype

int32_t d_decode_dba(
    DB_ADDR  dba, 
    FILE_NO *file, 
    F_ADDR8 *slot);
int32_t d_decode_ldba(
    const DB_ADDR dba,
    uint16_t     *uno,
    FILE_NO      *file, 
    F_ADDR8      *slot);

Parameters

dba (input) The database address to be decoded.
uno (output) The union number, only relevant with database union.
file (output) A pointer to a variable in which the file number will be returned.
slot (output) A pointer to a variable in which the slot number will be returned.

Description

This function extracts the file number and slot number from the specified database address, dba. The file number is returned in a variable pointed to by file. The slot number is returned in an variable pointed to by slot.

Optionally, the uno (union number) can be obtained. This will always be 0 unless a database union is being used.

Example

DB_ADDR        dba;   /* database address of current record */
short          file;  /* file number */
unsigned long  slot;  /* slot number */
...

/* record to be displayed is current record */
d_crget(&dba, task, CURR_DB);
d_decode_dba(dba, &file, &slot);
printf("id number: %d-%ld\n", (int) file, slot);
