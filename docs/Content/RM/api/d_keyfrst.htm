d_keyfrst

Find record with first key

Prototype

int32_t d_keyfrst(
    int32_t  FIELD, 
    DB_TASK *task, 
    int32_t  dbn);

Parameters

FIELD (input) The field type of the key field for which the first key will be found.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

Function d_keyfrst finds the record occurrence associated with the first occurrence of the key FIELD.

Currency Changes

curr_rec = record with first key;

Locking Requirements

Read lock on key file containing FIELD.

User's Guide

Data Retrieval Using Keys

Required Headers

#include "rdm.h"

Libraries

Library Name Description

Return Codes

Value Name Description
0 S_OKAY normal return, okay
2 S_NOTFOUND record not found
-17 S_NOTKEY field is not a valid key
-24 S_NOTLOCKED attempt to access unlocked set/record
-47 S_INVINDEX Invalid index type
