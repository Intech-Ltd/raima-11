d_trdeletemark

Remove a transaction mark

Prototype

int32_t d_trdeletemark(
    const char *mtag,
    DB_TASK    *task)

Parameters

mtag (input) A pointer to the marker within the transaction to remove.
task (input) A pointer to a DB_TASK structure.

Description

The d_trdeletemark is a function that allows any mark information to be removed (before the transaction is aborted or committed). This is useful in the scenario where marks are added many times within a transaction. If a sufficient number of them occur, memory gets wasted. Deleting marks when there is no longer a reason to keep the mark around can save memory because for each page modified since the tag (mark) was applied, a copy of the original page is kept to rollback to if necessary.

Currency Changes

None.

Locking Requirements

None.
