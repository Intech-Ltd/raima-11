exp_xmlschemarec

Export a record definition to XML Schema or DTD format

Prototype

int32_t exp_xmlschemarec(
    size_t    *nbytes, 
    char     **xmlschemabuf, 
    int32_t    format, 
    int32_t    rectype, 
    DB_EXPENV *env);

Parameters

nbytes (output) number of bytes, including null terminator, returned in xmlbuf – the caller may pass NULL if this result is not required
xmlschemabuf (output) buffer containing database dictionary represented in XML schema or DTD format – the caller may pass NULL if this result is not required
format (input) one of the following values, specifying the format of the schema:
DB_EXP_XML_SCHEMA -Generate an XML Schema
DB_EXP_XML_DTD -Generate DTD
rectype (input) record type whose definition is required
env (input) export state data

Description

This function generates a partial XML Schema or DTD, corresponding to the definition of record type rectype in the source database. It is identical to the function exp_xmlschema, except that it generates only the part of the XML Schema or DTD that applies to the specified record type.

As with exp_xmlschema, the UTF-8 encoding format is used.
