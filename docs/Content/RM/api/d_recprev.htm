d_recprev

Find previous occurrence of current record

Prototype

int32_t d_recprev(
    DB_TASK *task, 
    int32_t  dbn);

Parameters

task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This function will find the previous occurrenceOne record instance within a record type, specifically associated with record type scanning (d_recfrst, d_recnext, d_recprev, d_reclast), where the current occurrence of a record type is used to bookmark the position on a record type scan. Record occurrences are ordered by their physical appearance in a data file. The current occurrence is not the same as the current record, although the current record will also be set by the scanning functions. of the record of the same type as was found with a previous call to d_recprev, d_recfrst, d_reclast, d_recnext, or d_recset. This function provides sequential record access capability. The records are retrieved in descending database address order. Records of a different type or deleted records are skipped.

The S_NOTFOUND return code indicates that there are no more records in the database. If a subsequent d_recprev is executed, the last record in the database will be retrieved.

The S_NOTYPE return code indicates that there are no records in the database associated with the current record type.

This function establishes a new current occurrence for the record type which is a bookmark used for scanning. The current record and current occurrence are not the same, although this function will set them both.

Currency Changes

curr_rec = found record;

Locking Requirements

Allowed with no locks, but old or deleted data may be read. Use a read lock on current record type to ensure that current data is read.

Example

int32_t status;
int32_t info_total = 0;

/* Compute total number of info records in database */
for ( status = d_reclast(INFO, task, CURR_DB);
      status == S_OKAY; 
      status = d_recprev(task, CURR_DB))
{
    ++info_total;
}
printf("total of %d info records in database\n", info_total);
