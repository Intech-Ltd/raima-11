d_errorname

Returns mnemonic name for error or status code

Prototype

const char * d_errorname(
    int32_t  errnum);

Parameters

errnum (input) Status or error code returned by a core API function

Description

This function returns an error mnemonic associated with the given error or status code.

Required Headers

#include "rdm.h"

Libraries

Library Name Description
rdmrdm11 RDM Runtime Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Returns pointer to ASCII char string or NULL if invalid errnum is provided.

See Also

d_errorstr

#include "rdm.h"

        return retcode;
