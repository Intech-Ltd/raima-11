d_setdb

Set current database

Prototype

int32_t d_setdb(
    int32_t  dbn, 
    DB_TASK *task);

Parameters

dbn (input) The database number.
task (input) A pointer to a DB_TASK structure.

Description

This function sets the number of the current database to dbn. All subsequent calls to RDM functions that have a dbn of -1 (or CURR_DB) are applied to the specified database up until the next d_setdb call is executed. The number of the database is defined by the position of the database name in the database name list passed to the d_open and d_iopen functions. The first name has database number 0, the second has 1, and so on.

Note: Be sure to note the database number before the task parameter for this function. This is different from the standard order.

Example

int status;
status = d_open("genledg;acctsrec;acctspay", "s", task);
if ( status != S_OKAY ) {
    if ( status == S_UNAVAIL )
        printf("database(s) not currently available\n");
    exit(0);
}
