d_dbnum

Find current database number for a given database name

Prototype

int32_t d_dbnum(
    const char *dbname, 
    DB_TASK    *task);
int32_t d_dbnumW(
    const wchar_t *dbname,   /* UNICODE string */
    DB_TASK       *task);

Parameters

dbname (input) The database name for which the number is sought.
task (input) A pointer to a DB_TASK structure.

Description

This function will give the current database number for the given database name. If the database is not open, S_INVDB will be returned.

The string can contain a path as well as the database name. This is helpful when databases have the same name but different paths. If a path is included, it must be the exact path used in d_open. If no path is included, only the database name will be used even if d_open contained a path.

Example

int32_t my_dbn;

/* Open an initial set of databases */
d_open("db1;db2;db3", "s", task);
printf("Database db1 is number: %d", d_dbnum("db1"));

my_dbn = d_dbnum("db2", task);
d_keyfrst(KEY, task, my_dbn);

/* Close database db2 */
d_iclose(my_dbn, task);
