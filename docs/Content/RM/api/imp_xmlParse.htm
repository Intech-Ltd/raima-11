imp_xmlParse

Parse more of the document.

Prototype

int32_t imp_xmlParse(
    DB_IMPENV  *env, 
    const char *xmlbuf, 
    size_t      len, 
    int32_t     isFinal);

Parameters

env (input) Import session data
xmlbuf (input) Buffer containing XML text to be parsed.
len (input) Length of *xmlbuf in bytes.
isFinal (input) Set to TRUE is xmlbuf contains the end of the XML document.

Description

Parse some more of the XML document. The string xmlbuf is a buffer containing part (or perhaps all) of the document. The number of bytes of xmlbuf that are part of the document is indicated by len. This means that xmlbuf does not need to be null terminated. The isFinal parameter informs the parser that this is the last piece of the XML document. The last piece can be an empty string (len is zero).

User's Guide

XML Import

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-56 S_IMPBADXML bad xml data
-57 S_IMPBADDATA bad data

Example Code

d_opentask(&task);
imp_init(&env, 0, NULL, 0, 0, NULL, task);   
do {
    size_t len = fread(inputbuf, 1, sizeof(inputbuf), fp);
    done = len < sizeof(inputbuf);
    if (imp_xmlParse(env, inputbuf, len, done) != S_OKAY) {
        int32_t    errcnt, ii;
        DB_IMPERR *errlist;

        imp_xmlGetErrors(env, &errcnt, &errlist);
        for (ii=0; ii < errcnt; ++ii) {
            fprintf(stderr, "%s (%d) [Tag=%s] @ line %d\n",
            errlist[ii].text, errlist[ii].code, 
            errlist[ii].name, errlist[ii].line);
        }
        break;  /* exit loop */
    }
} while (!done);
