d_members

Get count of current set members

Prototype

int32_t d_members(
    int32_t  SET, 
    int32_t *count, 
    DB_TASK *task, 
    int32_t  dbn);

Parameters

SET (input) The set type for which the current membership count is to be returned.
count (output) A pointer to a integer variable that receives the total member count.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

Function d_members copies the total number of members for the current owner of SET into the integer pointed to by count.

Currency Changes

None.

Locking Requirements

Allowed with no locks, but may read old or deleted data. Use a read lock on SET to ensure current data.

Example

char name[32];   /* author name */
int32_t num;     /* number of pubs */
int32_t ret;
DB_TASK *task;
...
/* Print list of authors with total number of associated publications in database */
for (ret = d_findfm(AUTHOR_LIST, task, 0);
     ret == S_OKAY; 
     ret = d_findnm(AUTHOR_LIST, task, 0) )
{
    d_crread(NAME, name, task, 0);
    d_setor(HAS_PUBLISHED, task, 0);
    d_members(HAS_PUBLISHED, &num, task, 0);
    printf("Author %s has %ld publications on file.\n", name, num);
}
