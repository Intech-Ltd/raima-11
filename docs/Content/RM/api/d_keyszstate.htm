d_keyszstate

Retrieve the size of the key state buffer

Prototype

int32_t d_keyszstate(
    int32_t   keyfield,
    uint32_t *pSize,
    DB_TASK  *hDb,
    int32_t   dbn);

Parameters

keyfield (input) The key field type for which to get the state size.
pSize (output) A pointer to the size, in bytes, of the buffer to store key state information.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This function retrieves the size of a buffer to store state information for the specified key field, regardless of the key state.  The retrieved size does not change during execution of the application.  Your application uses the d_keyszstate function with d_keyrdstate and d_keywrstate to save and restore the state of a key file.

Required Headers

#include "rdm.h"

Libraries

Library Name Description
rdmrdm11 RDM Runtime Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-4 S_INVDB cannot open dictionary
-17 S_NOTKEY field is not a valid key
-52 S_INVTASK invalid task

Currency Changes

None

Locking Requirements

None

Example

#include "rdm.h"
#include "tims.h"

int32_t testKeyStateFcns(const char *id_code, DB_TASK *task)
{
  int32_t   stat;
  uint32_t  usSize;
  char      szIdCode[SIZEOF_ID_CODE];
  void     *pvBuf;

  if ((stat = d_reclock(INFO, "r", task, CURR_DB)) != S_OKAY)
    return stat;
    
  if ((stat = d_keyfind(ID_CODE, id_code, task, CURR_DB)) != S_OKAY) {
    d_recfree(INFO, task, CURR_DB);
    return stat;
  }

  d_keyszstate(ID_CODE, &usSize, task, CURR_DB);
  pvBuf = psp_getMemory(usSize, 0);
  
  d_keyrdstate(ID_CODE, pvBuf, task, CURR_DB);  /* read the key state */

  d_keyfrst(ID_CODE, task, CURR_DB);  /* change the key state on purpose */

  d_keywrstate(ID_CODE, pvBuf, task, CURR_DB);  /* restore the key state */

  while (d_keynext(ID_CODE, task, CURR_DB) == S_OKAY) {
    if ((stat = d_keyread(szIdCode, task)) == S_OKAY)
      fprintf(stdout, "%s\n", szIdCode);
  }

  psp_freeMemory(pvBuf, 0);
  d_recfree(INFO, task, CURR_DB);
  
  return stat;
}
