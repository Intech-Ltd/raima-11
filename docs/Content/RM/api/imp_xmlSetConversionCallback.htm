imp_xmlSetConversionCallback

Set custom data conversion callback.

Prototype

void imp_xmlSetConversionCallback(
    DB_IMPENV    *env, 
    IMPXMLCNVPROC convertfunc 
);

Parameters

env (input) Import session data
convertfunc (input) User defined custom data conversion processing function. The valid return values for the convertfunc are:
IMPXML_CONTINUE - Continue with default processing
IMPXML_BREAK -Skip default processing

User defined callback functions can replace or enhance the default behavior of the import function. The return code from the callback function signals the import to continue with the default processing or signal that processing is complete. The user defined callback functions must return one of the above return codes.

Description

The data conversion callback will be called prior to converting the input data and storing the data into the current record buffer. recbuf will be a pointer to the beginning of the current record buffer. fldptr will be a pointer to beginning of the current field in recbuf. fldconst is the field constant identifier. input is the data array to be converted. inputlen is the length of the null-terminated input buffer.

IMPXMLCNVPROC typedef

typedef int32_t (*IMPXMLCNVPROC)(DB_TASK *task, 
        int32_t fldconst,
        char   *recbuf,
        char   *fldptr,
        char   *input,
        size_t  inputlen,
        void   *userEnv);
