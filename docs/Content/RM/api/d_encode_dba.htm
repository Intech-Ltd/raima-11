d_encode_dba, d_encode_ldba

Encode database address

Prototype

int32_t d_encode_dba(
    FILE_NO  file, 
    F_ADDR8  slot, 
    DB_ADDR *dba);
int32_t d_encode_ldba(
    uint16_t uno,
    FILE_NO  file, 
    F_ADDR8  slot, 
    DB_ADDR *dba);

Parameters

uno (input) The union number, relevant only within a database union.
file (input) The file number of the database address to be formed.
slot (input) The slot number of the database address to be formed.
dba (output) A pointer to a variable to which the encoded database address will be returned.

Description

Function d_encode_dba forms a database address from file number file and slot number slot. The encoded database address is returned in a DB_ADDR variable pointed to by dba. The use of d_decode_dba is explained in the Direct Access Retrieval section of the RDM User's Guide.

Optionally, the uno (union number) can be explicitly set. This should always be 0 unless a database union is being used. The d_encode_dba form always sets it to 0.

Example

DB_ADDR          dba;       /* database address */
short            file;      /* file number */
unsigned long    slot;      /* slot number */
struct record    rec;       /* record buffer */
...
/* extract file number and slot number from ID number string */
get_numbers_from_user(&file, &slot);

/* form database address */
d_encode_dba(file, slot, &dba);

/* set current record and read record contents */
d_crset(&dba, task, CURR_DB);
d_recread(&rec, task, CURR_DB);
...
