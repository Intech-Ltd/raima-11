d_off_opt

Turn off runtime options

Prototype

int32_t d_off_opt(
    uint32_t optflag, 
    DB_TASK *task);

Parameters

optflag (input) The option flag containing bit settings for the options to be turned off.
task (input) A pointer to a DB_TASK structure.

Description

This function is used to turn off any or all RDM runtime options. Several runtime options are currently controlled using this function and can be found in Runtime Options.

The optflag parameter contains a bit status integer that has a bit associated with each option. The rdm.h header file contains constant definitions for each option.
