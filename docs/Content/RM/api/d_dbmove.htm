d_dbmove

Move database to a new sub-directory under the docroot.

Prototype

int32_t d_dbmove(
    const char *dirname, 
    DB_TASK    *task, 
    int32_t     dbn);
int32_t d_dbmoveW(
    const wchar_t *dirname,   /* UNICODE string */
    DB_TASK       *task, 
    int32_t        dbn);

Parameters

dirname (input) New sub-directory.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

Move the database identified by dbn from the current directory location of the database to dirname. The dirname parameter must be specified as a path relative to the TFS docroot. It must not be specified as an absolute path and must not contain a database name. If dirname is NULL or ".", the database is moved to the TFS docroot.

Database must be opened in exclusive mode ('x'), must not be a union, and cannot have an active transaction.

Return Codes

Number of database (0 - ...).

Value Name Description
-1 S_DBOPEN database not opened
-26 S_TRACTIVE transaction already active
-32 S_EXCLUSIVE exclusive access required
-51 S_INVFORUNIONDB operation invalid for a union database
-904 S_NOMEMORY out of memory
