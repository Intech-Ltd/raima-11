d_setkey

Set value of key field for d_makenew

Prototype

int32_t d_setkey(
    int32_t     FIELD, 
    const void *value, 
    DB_TASK    *task, 
    int32_t     dbn);

Parameters

FIELD (input) The field type of the key field for which the value is to be stored.
value (input) A pointer to the value of the field to be stored.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This function must be called for each key field in a record that will subsequently be created by a call to d_makenew. This establishes the data content of all key fields defined in the record so that the system can create those keys when d_makenew is called. Optional and compound key fields cannot be processed by d_setkey.

Currency Changes

None.

Locking Requirements

None.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-5 S_INVFLD invalid field name
-17 S_NOTKEY field is not a valid key
-39 S_COMKEY record/field has/in a compound key
-41 S_INVKEY invalid optional key - key cannot be used in d_setkey
