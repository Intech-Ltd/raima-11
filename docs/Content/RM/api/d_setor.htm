d_setor

Set current owner from current record

Prototype

int32_t d_setor(
    int32_t  SET, 
    DB_TASK *task, 
    int32_t  dbn);

Parameters

SET (input) The set type for which the current record will be assigned as current owner.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This function assigns the current owner of SET from the current record. The current record must be a legal owner of SET. The current member of SET will become NULL_DBA.

Currency Changes

curr_own[SET] = curr_rec;

curr_mem[SET] = NULL_DBA;

Locking Requirements

Allowed with no locks, but may read old or deleted data. Use a read lock on SET to ensure current data.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-2 S_INVSET invalid set
-7 S_NOCR no current record
-18 S_INVOWN record not legal owner of set

    }
}
