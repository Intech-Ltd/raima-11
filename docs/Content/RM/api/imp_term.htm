imp_term

Resolve forward references, remove temporary databases and free memory associated parser and user data environment.

Prototype

int32_t imp_term(
    DB_IMPENV *env);

Parameters

env (input) Import session data

Description

The forward reference database, if it exists, will be processed. The temporary database files will be closed and removed. Memory allocated by the imp_init function, including the env pointer, will be freed.
