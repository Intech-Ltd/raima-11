d_closetask

Close a database task

Prototype

int32_t d_closetask(
    DB_TASK *task);

Parameters

task (input) A pointer to a DB_TASK structure.

Description

This function will free the memory associated with a database task. The d_opentask function must previously have created the task, and any databases opened within the task must first be closed.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-48 S_DBCLOSE database not closed

See Also

d_opentask

Example

DB_TASK task;

d_opentask(&task);
d_open("db", "o", &task);
d_makenew(REC, &task, CURR_DB);
d_close(&task);
d_closetask(&task);
