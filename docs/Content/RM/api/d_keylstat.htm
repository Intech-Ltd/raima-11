d_keylstat

Get lock status of key type

Prototype

int32_t d_keylstat(
    int32_t  FIELD, 
    char    *type, 
    DB_TASK *task, 
    int32_t  dbn);
int32_t d_keylstatW(
    int32_t  FIELD, 
    wchar_t *type, 
    DB_TASK *task, 
    int32_t  dbn);

Parameters

FIELD (input) The field type of the key field that will be locked.
type (output) A pointer to a character variable that contains the current lock type status.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This function returns the calling process's current lock status for key field FIELD in the character variable pointed to by type. The lock types which can be returned are as follows:

  • 'r' (read lock)
  • 'w' (write or keep lock)
  • 's' (static record)
  • 'f' (free; exclusive access mode always returns an 'f' status)
