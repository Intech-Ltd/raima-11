d_setnum

Returns set table index from #define and database number

Prototype

int32_t d_setnum(
    int32_t *index, 
    int32_t  SET, 
    DB_TASK *task, 
    int32_t  dbn);

Parameters

index (output) The index into the set table for SET.
SET (input) The set type for which to get the index.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This function takes SET (the #define for sets (from ddlp) and the database number and returns the set table index for that set. The index number can then be used with functions like d_internals.

Currency Changes

None. (If an error occurs, the output buffer is not changed.)

Locking Requirements

None.

Required Headers

#include "rdm.h"

Libraries

Library Name Description
rdmrdm11 RDM Runtime Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-2 S_INVSET invalid set

See Also

d_opentask

d_open, d_open_ptr, d_iopen, d_iopen_ptr

Example

#include "rdm.h"
#include "internal.h"

{
    SET_ENTRY      setEntry;
    int32_t        setIndex;
    ...
    d_open("tims", "s", task);
   
   
    /* get info about the set type */
    d_setnum(&setIndex, HAS_PUBLISHED, task, CURR_DB);
    d_internals(task, TOPIC_SET_TABLE, 0, setIndex, 
        &setEntry, sizeof(SET_ENTRY));
    ...
}
