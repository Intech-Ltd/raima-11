d_blobsize

Retrieve the size of BLOB data

Prototype

int32_t d_blobsize(
    int32_t   field,
    uint32_t *size,
    DB_TASK  *task, 
    int32_t   dbn);

Parameters

field (input) The field type to access from the current record. This must be a BLOB field.
size (input) A pointer to the size, in bytes, of the BLOB data.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This function retrieves the size of the specified BLOB data associated with the indicated field identifier (field) in the current record.

Currency Changes

None.

Locking Requirements

Read lock on record type of current record.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
2 S_NOTFOUND record not found
6 S_DELETED record/set deleted since last accessed
-7 S_NOCR no current record
-24 S_NOTLOCKED attempt to access unlocked set/record
-926 S_NOTBLOB the specified field is not a blob field

Example

int16_t readBlobFromStart(int32_t, void *, uint32_t, DB_TASK);

int16_t testReadBlob1(int32_t nBlobId, DB_TASK task)
{
    int16_t  stat;
    void    *pvBuffer;
    uint32_t uSize;

    if ((stat = d_blobsize(nBlobId, &uSize, task, CURR_DB)) == S_OKAY) {
        pvBuffer = psp_getMemory(uSize, 0);
        if (pvBuffer) {
            stat = readBlobFromStart(nBlobId, pvBuffer, uSize, task, CURR_DB);
            if (stat == S_OKAY) {
                /* process the blob data */
            }

            psp_freeMemory(pvBuffer, 0);
        }
    }

    return stat;
}
