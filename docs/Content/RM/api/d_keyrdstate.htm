d_keyrdstate

Read the key state information

Prototype

int32_t d_keyrdstate(
    int32_t  keyfield,
    void    *buf,
    DB_TASK *task,
    int32_t  dbn);

Parameters

keyfield (input) The key field type for which to read the state.
buf (output) A pointer to the key state information.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This function copies state information for the position of the specified key into the buffer indicated by buf.  Your application uses the d_keyrdstate function with d_keyszstate and d_keywrstate to save and restore the current position of a key in a key file.

Required Headers

#include "rdm.h"

Libraries

Library Name Description
rdmrdm11 RDM Runtime Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-4 S_INVDB cannot open dictionary
-17 S_NOTKEY field is not a valid key
-52 S_INVTASK invalid task

Currency Changes

None

Locking Requirements

None
