d_setom

Set current owner from current member

Prototype

int32_t d_setom(
    int32_t  O_SET, 
    int32_t  M_SET, 
    DB_TASK *task, 
    int32_t  dbn);

Parameters

O_SET (input) The set type whose current owner will be assigned from the current member of M_SET.
M_SET (input) The set type whose current member will become the current owner of O_SET.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This function assigns the current owner of O_SET from the current member of M_SET. The current member of M_SET must be a legal owner of O_SET. The current member of O_SET will become NULL_DBA.

If the current member of M_SET is deleted, this function will detect it and fail with a return of S_DELETED.

Currency Changes

curr_own[O_SET] = curr_mem[M_SET];

curr_mem[O_SET] = NULL_DBA;

Locking Requirements

None.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
6 S_DELETED record/set deleted since last accessed
-2 S_INVSET invalid set
-9 S_NOCM set has no current member
-18 S_INVOWN record not legal owner of set

Example

/* disconnect and delete intersect and (possibly) key word */
d_setom(INFO_TO_KEY, HAS_PUBLISHED, task, CURR_DB);
while ( d_findnm(INFO_TO_KEY, task, CURR_DB) == S_OKAY )
{
   d_discon(INFO_TO_KEY, task, CURR_DB);
   d_setmr(KEY_TO_INFO, task, CURR_DB);
   d_discon(KEY_TO_INFO, task, CURR_DB);
   d_delete(task, CURR_DB);
   d_members(KEY_TO_INFO, &count, task, CURR_DB)
   if ( count == 0L )
   {
      	/* delete key word */
      	d_setro(KEY_TO_INFO, task, CURR_DB);
      	d_delete(task, CURR_DB);
   }
}
