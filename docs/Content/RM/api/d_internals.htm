d_internals

Access internal RDM data

Prototype

int32_t d_internals(
    DB_TASK* task, 
    int32_t  topic, 
    int32_t  id, 
    int32_t  elem, 
    void    *ptr, 
    uint32_t size);

Parameters

task (input) A pointer to a DB_TASK structure.
topic (input) The topic of data requested.
id (input) The identifier of a selection within the TOPIC_GLOBALS topic.
elem (input) The element of a table, if topic is a table.
ptr (output) A pointer to the area to copy the data.
size (input) The size *ptr in bytes.

Description

NOTE: This function is subject to changes due to implementation changes between versions. Details of the data structures are only available to those who own source code. It is unadvisable to become dependent on information returned by this function.

This function copies a data item from the topic (an RDM internal table), id (a subtopic within the table), and elem (an index into the table), to ptr, the pointer supplied for the size given.

The topics (shown below) and IDs are provided in the file internal.h.

Topic Elem Ptr Type Description
TOPIC_DB_TABLE dB No DB_ENTRY  
TOPIC_FIELD_NAMES Field No char * Returns pointer to field name designated by the Field No provided in elem. The Field No is a global identifier can be derived using the d_fldnum function.
TOPIC_FIELD_TABLE Field No FIELD_ENTRY Returns pointer to field information designated by the Field No provided in elem. The Field No is a global identifier can be derived using the d_fldnum function.
TOPIC_FILE_TABLE File No FILE_ENTRY Returns pointer to file name designated by the File No provided in elem. The first File No is located at ft_offset (see TOPIC_DB_TABLE).
TOPIC_GLOBALS
TOPIC_KEY_TABLE id KEY_ENTRY
TOPIC_MEMBER_TABLE id

MEMBER_ENTRY
TOPIC_RECORD_NAMES Rec No char * Returns pointer to field name designated by the Record No provided in elem. The first Record No is located at rt_offset (see TOPIC_DB_TABLE).
TOPIC_RECORD_TABLE Rec No RECORD_ENTRY Returns pointer to record information designated by the Record No provided in elem. The first Record No is located at rt_offset (see TOPIC_DB_TABLE).
TOPIC_SET_NAMES Set No char * Returns pointer to set name designated by the Set No provided in elem. The first set for a database is located at st_offset (see TOPIC_DB_TABLE).
TOPIC_SET_TABLE Set No SET_ENTRY Returns pointer to set information designated by the Set No provided in elem. The first Set No is located at st_offset (see TOPIC_DB_TABLE).
TOPIC_SORT_TABLE id SORT_ENTRY
TOPIC_TASK TASK Copy the current task structure.

The table topics do not use an ID, but need an element.

The TOPIC_GLOBALS must use one of the following IDs but they do not need an element.

Sub-Topic ID Elem Ptr Type Description
ID_DB_TIMEOUT uint32_t
ID_DBUSERID char * Unique DBUSERID either set by d_dbuserid or automatically generated.
ID_NO_OF_DBS uint16_t Number of databases open in this task.
ID_READLOCKSECS uint32_t Number of seconds before timeout when requesting a read lock. Set by d_locktimeout.
ID_SIZE_FD Size of the FIELD_TABLE of the current database.
ID_SIZE_FT Size of the FILE_TABLE of the current database.
ID_SIZE_KT Size of the KEY_TABLE of the current database.
ID_SIZE_MT Size of the MEMBER_TABLE of the current database.
ID_SIZE_RT Size of the RECORD_TABLE of the current database.
ID_SIZE_SRT Size of the SORT_TABLE of the current database.
ID_SIZE_ST Size of the SET_TABLE of the current database.
ID_TRANS_ID char * Transaction ID as set by d_trbegin.
ID_WRITELOCKSECS int32_t Number of seconds before timeout when requesting a write lock. Set by d_locktimeout.

This data is not required by most applications. Its interpretation by the programmer requires knowledge of the RDM internals. A good source of information (such as structure definitions) can be found in the dbtype.h header file.

Example

#include <stdio.h>
#include "rdm.h"
#include "internal.h"
#include "tims.h"


main()
{
    int32_t no_of_files, i;
    FILE_ENTRY fEntry;    /* defined in dbtype.h */
    DB_TASK *task;

    /* Open the tims database in one user mode */
    d_opentask(&task);
    d_open("tims", "o", task);

    /* Get the number of files in the file table. */
    d_internals(task, TOPIC_GLOBALS, ID_SIZE_FT, 0, &no_of_files,
                sizeof(no_of_files));

    /* Get the file names from the internal file table. ".ft_name" is a member */
    for (i = 0; i < no_of_files; i++) {
        d_internals(task, TOPIC_FILE_TABLE, 0, i, &fEntry, sizeof(fEntry));
        printf("File name is %s\n", fEntry.ft_name);
    }

    d_close(task);            /* Close the database */
    d_closetask(task);
}
