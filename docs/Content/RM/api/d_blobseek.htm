d_blobseek

Seek to a given position in the BLOB data

Prototype

int32_t d_blobseek(
    int32_t   field,
    uint32_t  offset,
    SEEK_TYPE origin,
    DB_TASK  *task, 
    int32_t   dbn);

Parameters

field (input) The field type to access from the current record. This must be a BLOB field.
offset (input) The byte offset in the BLOB data. The offset is the number of bytes from the point of origin (origin parameter).
origin (input) The origin indicator for the seek operation. To place the origin at the current position, specify B_CURRENT. To place the origin at the beginning or end of the BLOB data, specify B_START or B_END, respectively.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This function adjusts the current byte offset in BLOB data according to the specified point of origin. If the origin parameter is set to B_START, the offset must be greater than or equal to zero and d_blobseek sets the position from the start of the BLOB data. If origin contains B_CURRENT, d_blobseek sets the position to offset bytes from the current position. Finally, if origin contains B_END, the offset must be less than or equal to zero, and d_blobseek sets the position from the end of the BLOB data. The d_blobseek function returns S_BLOBPOS if the offset sets the position beyond the boundaries of the BLOB data.

Currency Changes

None.

Locking Requirements

Read lock on record type of current record.

Example

int16_t readBlobFromStart(
    int32_t nBlobId,
    void *pvBuffer,
    uint32_t uLen,
    DB_TASK task)
{
    int16_t  stat;
    uint32_t uOutlen;

    /* move the pointer to the beginning of the blob data */
    if ((stat = d_blobseek(nBlobId, 0, B_START, task, CURR_DB)) == S_OKAY)
{
    /* read the blob data from the beginning */
    stat = d_blobread(nBlobId, pvBuffer, uLen, &uOutlen, task, CURR_DB);
}

  return stat;
}
