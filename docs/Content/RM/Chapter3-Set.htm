Summary Listing of Set Relationship Functions

The following table lists alphabetically all the set relationship functions, along with brief descriptions. All functions have int32_t as their return types.

Currency Manipulation Functions

Currency Manipulation Functions
Function Description
d_crset Set database address of current record from variable.
d_csmset Set database address of current member of SET from variable.
d_csoset Set database address of current owner of SET from variable.
d_setro Set current record from current owner of SET.
d_setrm Set current record from current member of SET.
d_setor Set current owner of SET from current record.
d_setom Set current owner of SET1 from current member of SET2.
d_setoo Set current owner of SET1 from current owner of SET2.
d_setmr Set current member of SET from current record.
d_setmo Set current member of SET1 from current owner of SET2.
d_setmm Set current member of SET1 from current member of SET2.

Currency Access Functions

Currency Access Functions
Function Description
d_crget Get database address of current record.
d_csmget Get database address of current member of SET.
d_csoget Get database address of current owner of SET.

Currency Table Functions

Currency Table Functions
Function Description
d_rdcurr Read the currency table into buffer.
d_rerdcurr Reread the currency table into existing buffer.
d_wrcurr Write the currency table from buffer and free buffer.
d_keyrdstate Read the key state information
d_keyszstate Retrieve the size of the key state buffer
d_keywrstate Restore the key state information

Key Access Functions

Key Access Functions
Function Description
d_keyexist Determine if optional key has been stored.
d_keyfrst Find record with first KEY.
d_keylast Find record with last KEY.
d_keyfind Find first record with exactly matching KEY value.
d_keynext Find record with next KEY.
d_keyprev Find record with previous KEY.
d_keyread Read contents of last key scanned.
d_curkey Set current keys from current record.
d_keydir Set the search direction for a key

Set Navigation Functions

Explicit set the current owner
Function Description
d_setor Set current owner from current record
d_setoo Set current owner from current owner
d_setom Set current owner from current member
d_csoset Set the current owner to point to record given by database address
d_findco Find current owner of current record

Explicit set the current member (and implicit set the current owner)
Function Description
d_setmr Set current member from current record
d_setmo Set current member from current owner
d_setmm Set current member from current member
d_csmset Set the current member to point to record given by database address

Normal member navigation
Function Description
d_findfm Find first member of set
d_findlm Find last member of set
d_findnm Find next member of set
d_findpm Find previous member of set

Establishing Set Relationships

Set relationships can be established using the following functions:

Function Description
d_connect Connect current record to set
d_discon Disconnect current member from set
d_disdel Disconnect current member from set and delete current member

Update Current Member or Owner

Fields in current owner or current member records can be updated using the following functions:

Function Description
d_csmwrite Write data to field of current member
d_csowrite Write data to field of current owner

Query Current Member or Owner

Current owner or current member can be queried using the following functions:

Function Description
d_members Get count of current set members
d_cotype Get record type of current owner
d_cmtype Get record type of current member
d_csoget Get database address of current owner
d_csmget Get database address of current member
d_setro Set current record from current owner
d_setrm Set current record from current member
d_csoread Read data from field of current owner
d_csmread Read data from field of current member

Saving and restoring the currency

Function Description
d_rdcurr Read currency tables
d_rerdcurr Reread currency tables
d_wrcurr Write currency tables

Other set related functions

The current record can be queried for set relationships using the following functions:

Function Description
d_isowner Determine whether the current record is the owner of a set
d_ismember Determine whether the current record is a member of a set
