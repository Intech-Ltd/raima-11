Summary Listing of Basic Record and Key Functions

The following table lists all the basic record and key functions, along with brief descriptions. Please see Summary Listing of Set Relationship Functions for set relationship functions and Summary Listing of Core BLOB Functions for BLOB functions. All functions have int32_t as their return types.

Record Navigation

Records can be navigated by use of the following functions specified below:

Currency Manipulation Functions
Function Description
d_crset Set database address of current record from variable.
d_csmset Set database address of current member of SET from variable.
d_csoset Set database address of current owner of SET from variable.
d_setro Set current record from current owner of SET.
d_setrm Set current record from current member of SET.
d_setor Set current owner of SET from current record.
d_setom Set current owner of SET1 from current member of SET2.
d_setoo Set current owner of SET1 from current owner of SET2.
d_setmr Set current member of SET from current record.
d_setmo Set current member of SET1 from current owner of SET2.
d_setmm Set current member of SET1 from current member of SET2.

 

Basic record navigation
Function Description
d_recfrst Find first occurrence of record type
d_recnext Find next occurrence of record type
d_recprev Find previous occurrence of record type
d_reclast Find last occurrence of record type

Normal key navigation
Function Description
d_keyfind Find record by key
d_keyfrst Find record with first key
d_keynext Find record with next key
d_keyprev Find record with previous key
d_keylast Find record with last key
d_pkeyfind Find a record by its partial key
d_pkeynext Find the record occurrence associated with the next key
d_pkeyprev Find the record occurrence associated with the previous key

Creating and Deleting Records

Creating and deleting records can be performed using the following functions:

Function Description
d_fillnew Create and fill new record
d_makenew Make a new empty record
d_setkey Set value of key field for d_makenew
d_disdel Disconnect and delete current record
d_delete Delete current record

Query Current Record

Function Description
d_recread Read contents of current record
d_crread Read data from field of current record
d_crtype Get record type of current record
d_crget Get database address of current record
d_setor Set current owner from current record
d_setmr Set current member from current record
d_keyexist Determine whether an optional key exists
d_isowner Determine whether current record is the owner of a set
d_ismember Determine whether current record is member of a set

Update Current Record

Function Description
d_recwrite Write contents of current record
d_crwrite Write data to field in current record
d_keystore Store optional key
d_keydel Delete optional key
d_connect Connect current record to set
d_discon Disconnect current member from set

Saving and restoring the currency

Currency Table Functions
Function Description
d_rdcurr Read the currency table into buffer.
d_rerdcurr Reread the currency table into existing buffer.
d_wrcurr Write the currency table from buffer and free buffer.
d_keyrdstate Read the key state information
d_keyszstate Retrieve the size of the key state buffer
d_keywrstate Restore the key state information
