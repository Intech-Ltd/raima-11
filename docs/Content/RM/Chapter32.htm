Unicode Prototypes

The RDM product supports Unicode. Functions that have character string parameters have parallel functions that accept Unicode character strings. The function name is given a suffix of 'W', and all 'char' type parameters are 'wchar'. For example, the prototype:

int32_t EXTERNAL_FCN d_open(
    const char *dbnames,
    const char *openmode,
    DB_TASK    *task);

has a corresponding Unicode version with the following prototype:

int32_t EXTERNAL_FCN d_openW(
    const wchar_t *dbnames,
    const wchar_t *openmode,
    DB_TASK       *task);
