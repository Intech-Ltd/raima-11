psp_threadId

Retrieve current thread's ID.

Prototype

psp_thread_t psp_threadId(
    void);

Parameters

None.

Description

This function returns the current thread's ID.

Required Headers

#include "psp.h"

Libraries

Library Name Description
rdmpsp11 Platform Support Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Thread ID value.

Example

#include "psp.h"

...
void check_id(void)
{
    printf("Thread ID: %u\n", psp_threadId()); 
}
