psp_fileSeekWrite

Write to a specified location in a file

Prototype

uint32_t psp_fileSeekWrite(
    PSP_FH           handle,
    PSP_FILE_OFFSET  addr,
    const void      *buf,
    uint32_t         size);

Parameters

handle (input) The handle to the PSP file opened with psp_fileOpen.
addr (input) The position/offset within the file at which to begin writing
buf (input) A pointer to the buffer containing the data to be written
size (input) The size, in bytes, of the block to be written

Description

This function writes a block of bytes from a buffer into a PSP file, beginning at the specified byte offset position.

Required Headers

#include "psp.h"

Libraries

Library Name Description
rdmpsp11 Platform Support Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Number of bytes written or -1 if write error occurred
