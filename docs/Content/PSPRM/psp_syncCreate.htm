psp_syncCreate

Create a synchronization object                   

Prototype

PSP_SEM psp_syncCreate (
    int16_t     type);

Parameters

type (input) The semaphore type indicator.  Specify one of these indicators:

PSP_MUTEX_SEM - A mutex semaphore. This semaphore is used to allow a single thread exclusive access to a data structure.
PSP_EVENT_SEM - An event semaphore.  This semaphore is used to allow multiple threads to wait for a single event.

Description

This function creates a synchronization object.  Two types of synchronization objects (semaphores) are available: event and mutex (see the type parameter above). The initial state for an event semaphore is "signaled".

Return Codes

The synchronization object handle to be used in subsequent psp_sync* calls, NULL if operation fails

See Also (Mutex)

psp_syncEnterExcl

psp_syncExitExcl

See Also (Event)

psp_syncResume

psp_syncStart

psp_syncWait

Example

#include "psp.h"

void testSyncCreate(PSP_SEM *pmSem, PSP_SEM *peSem)
{ 
    *pmSem = psp_syncCreate(PSP_MUTEX_SEM);
    *peSem = psp_syncCreate(PSP_EVENT_SEM);
    if (*pmSem == NULL || *peSem == NULL)
        exit(1);
    return;
}
