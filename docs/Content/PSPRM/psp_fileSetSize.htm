psp_fileSetSize

Set the current size of the file

Prototype 

int32_t psp_fileSetSize(
    PSP_FH          handle,
    PSP_FILE_OFFSET size);

Parameters

handle (input) The handle to the PSP file opened with psp_fileOpen.
size (input) The new size of the file.

Description

This function changes the size of the given file. If the new file size is larger than the old file size, NULLs are appended to the end. If the new file size is smaller than the old file size, the file is truncated.

Required Headers

#include "psp.h"
