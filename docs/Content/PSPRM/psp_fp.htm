psp_fprintf, psp_fputc, psp_fputs, psp_vfprintf

Write character(s) to a standard I/O stream

Prototype

int32_t psp_fprintf(
    PSP_FILE_STREAM stream,
    const char     *format,
    ...);
int32_t psp_fputc(
    int32_t         c,
    PSP_FILE_STREAM stream);
int32_t psp_fputs(
    const char     *str,
    PSP_FILE_STREAM stream);
int32_t psp_vfprintf(
    PSP_FILE_STREAM stream,
    const char     *format,
    va_list         args);

Parameters

stream (input) The handle to the PSP_FILE_STREAM file opened with psp_fopen or psp_fopen_std.
format (input) A printf-style format string.
c (input) A UTF-8 character.
str (input) A UTF-8 string.
... (input) Values to be printed.
args (input) Variable argument list (stdarg format).

Description

These functions correspond to the standard C library functions (minus the "psp_" prefix). The additional parameter, stream, must be a handle returned from psp_fopen or psp_fopen_std.

Required Headers

#include "psp.h"

Libraries

Library Name Description
rdmpsp11 Platform Support Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Number of bytes written or -1 if write error occurred
