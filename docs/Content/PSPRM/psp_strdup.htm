psp_strdup

Allocate a duplicate string

Prototype

void *psp_strdup(
    const char *str,
    PSP_MEMTAG  tag);

Parameters

str (input) The string to duplicate.
tag (input) The memory tag. To use a default system tag, specify NULL.

Description

This function allocates a new string and associates it with a memory tag. The contents of the string are copied from str. The length of the new string is strlen(str)+1.

If there is insufficient memory available, program control passes to the location where the jump buffer associated with the tag is set. If there is no active jump buffer, the function returns NULL.

A jump buffer can be associated with a memory tag in the psp_createTag call creating that tag. To change the jump buffer associated with a tag, call psp_resetTag.

Required Headers

#include "psp.h"

Libraries

Library Name Description
rdmpsp11 Platform Support Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

A pointer to the allocated block, or NULL.

Example

#include "psp.h"

void allocBuffers(
    char     **ppszInBuff,
    char     **ppszOutBuff,
    int        buffSize,
    PSP_MEMTAG mTag)
{ 
    /* allocate input buffer for 1 dump block */
    *ppszInBuff = psp_getMemory(buffSize, mTag);

    /* fill the buffer */
    getNextBuff(*ppszInBuff);

    /* create a copy of the input before changing it */
    *ppszOutBuff = psp_strdup(*ppszInBuff, mTag);
}
