psp_fileSync

Synchronize writes to a file

Prototype 

uint32_t psp_fileSync(
    PSP_FH handle);

Parameters

handle (input) The handle to the PSP file opened with psp_fileOpen.

Description

This function commits all writes on a PSP file to disk. All pending writes for this handle are guaranteed to be on the disk once the function has returned. This function has no effect on a file opened with PSP_FLAG_INMEMORY or PSP_FLAG_NO_SYNC.

Required Headers

#include "psp.h"
