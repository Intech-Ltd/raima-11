psp_fopen

Open a streaming file.

Prototype

PSP_FILE_STREAM psp_fopen(
    const char   *name,
    const char   *mode,
    PSP_FILE_MODE fmode);
PSP_FILE_STREAM psp_fopen_std(
    const char *std_name);

Parameters

name (input) The file name including the directory path.
mode (input) Open mode. Read, Write, Append. See below.
fmode (input) File mode. Unknown, UTF-8 or UTF-16. See below.

Description

This function opens a file and returns a handle of type PSP_FILE_STREAM. Interprets or adds a Byte-Order-Mark (BOM) when necessary. If the file cannot be opened, NULL is returned.

The psp_fopen_std function expects the name of a standard I/O stream, "stdin", "stdout" or "stderr".

Open Modes

Mode string Description
"a" Append. Will create the file if it doesn't exist.
"r" Read only.
"w" Write only. Will create new file if it doesn't exist. Will truncate an existing file.
"r+" Read/write mode. Will not truncate an existing file.
"w+" Read/write mode. Will create new file if it doesn't exist. Will truncate an existing file.

File Modes

Mode Description
PSP_FILE_MODE_UNKNOWN When opening a file to read, set from BOM in file, else UTF-8. When opening to write, set to UTF-8.
PSP_FILE_MODE_UTF8 When opening to write, set to UTF-8.
PSP_FILE_MODE_UTF16 When opening to write, set to UTF-16.

Required Headers

#include "psp.h"

Libraries

Library Name Description
rdmpsp11 Platform Support Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

None.

Example

#include "psp.h"
#include <fcntl.h>


int16 OpenFiles(
    const char      *inFileName,  /* IN:  input file name     */
    const char      *outFileName, /* IN:  output file name    */
    PSP_FILE_STREAM *phInFile,    /* OUT: input file pointer  */
    PSP_FILE_STREAM *phOutFile)   /* OUT: output file pointer */
{
    /* open files */
    *phInFile = psp_fopen(inFileName, "r", PSP_FILE_MODE_UNKNOWN);
    *phOutFile = psp_fopen(outFileName, "w+", PSP_FILE_MODE_UTF8);

    if (*phInFile == NULL || *phOutFile == NULL)
        return PSP_FAILED;

    return PSP_OKAY;
}
