psp_syncExitExcl

Exit an exclusive access section 

Prototype

void psp_syncExitExcl (
    PSP_SEM handle)

Parameter

handle (input) The handle to a mutex semaphore.

Description

This function exits an exclusive access section controlled by a mutex semaphore.  The next thread that is waiting on a call to psp_syncEnterExcl will be given control.

Return Codes

None

See Also

psp_syncEnterExcl

Example

#include "psp.h"

extern int blockCount;

void checkCount(PSP_SEM countSem)
{
    /* update count of dumped blocks */
    psp_syncEnterExcl(countSem);
    blockCount++;
    psp_syncExitExcl(countSem);
}
