psp_fileOpen

Open a PSP resource file.

Prototype

PSP_FH psp_fileOpen(
    const char *name,
    uint32_t    flags,
    uint16_t    xflags);
PSP_FH psp_fileOpenW(
    const wchar_t *name,  /* UNICODE string */
    uint32_t       flags,
    uint16_t       xflags);

Parameters

name (input) The file name including the directory path.
flags (input) A bit map of file open characteristics. Specify a combination of either O_BINARY or O_TEXT, one of O_RDONLY, O_RDWR, or O_WRONLY (O_RDONLY is the default), and any combination of O_APPEND, O_CREATE, and O_TRUNC. These flags are described below.
xflags (input) A bit map of extended file open characteristics of PSP resource file operation. Specify any combination of the flags listed below.

Description

This function opens a file. An RDM resource manager (PSP) file handle pointer is returned. If the file cannot be opened, NULL is returned. You should only use psp_fileSeekRead and psp_fileSeekWrite to perform the I/O operations on shared files.

File Open Characteristics Table (flags)

Flag Name Description
O_BINARY Binary file. No interpretation is performed.
O_TEXT Text file. The end-of-file marker is <Ctrl+D> (UNIX) or <Ctrl+Z>+<Enter> (Windows).
O_RDONLY Read-only mode. Write requests result in errors.
O_RDWR Read/write mode.
O_WRONLY Write-only mode.
O_APPEND Append mode. New writes are added to the end of the file.
O_CREAT Create mode. A new file is created. If the file already exists and O_TRUNC and O_APPEND are not specified, an error is returned.
O_TRUNC Truncate mode. Implies O_CREAT

Extended File Open Characteristics Table (xflags)

Flag Name Description
PSP_FLAG_DENYNO (Windows) Allow others to read or write this file
PSP_FLAG_DENYRD (Windows) Deny others from reading this file
PSP_FLAG_DENYWR (Windows) Deny others from writing to this file
PSP_FLAG_DENYRW (Windows) Deny others from reading or writing this file
PSP_FLAG_INMEMORY File is in-memory
PSP_FLAG_NO_SYNC Do not sync file to disk (disables fsync)

Required Headers

#include "psp.h"

Libraries

Library Name Description
rdmpsp11 Platform Support Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

None.

See Also

psp_fileClose

Example

#include "psp.h"
#include <fcntl.h>


int16 OpenFiles(
const char *inFileName,    /* IN:  input file name     */
const char *outFileName,   /* IN:  output file name    */
PSP_FH     *phInFile,      /* OUT: input file pointer  */
PSP_FH     *phOutFile)     /* OUT: output file pointer */
{
    /* open files */
    *phInFile = psp_fileOpen(inFileName, O_RDONLY|O_BINARY, PSP_DENYNO);
    *phOutFile = psp_fileOpen(outFileName, O_CREAT|O_RDWR|O_BINARY, PSP_DENYRD);

    if (*phInFile == NULL || *phOutFile == NULL)
        return PSP_FAILED;

    return PSP_OKAY;
}
