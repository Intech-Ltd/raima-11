psp_init

Initialize the platform support package system

Prototype

int32_t psp_init(
    void);

Parameters

none.

Description

This function initializes the PSP (Platform Support Package) system. It is required before any PSP function is called. This function may be called multiple times, but for each call to psp_init, there must be a closing call to psp_term(it only initializes and cleans up once).

Required Headers

#include "psp.h"

Libraries

Library Name Description
rdmpsp11 Platform Support Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Value Name Description
0 PSP_OKAY A normal return; successful.
2 PSP_FAILED  

Example

#include "psp.h"

psp_init();
...
/* use the psp functions */
...
psp_term();
