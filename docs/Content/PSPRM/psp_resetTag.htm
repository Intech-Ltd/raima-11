psp_resetTag

Reset the jump buffer associated with a tag

Prototype

void psp_resetTag(
    const jmp_buf  errloc,
    PSP_MEMTAG     tag);

Parameters

errloc (input) The jump buffer for an out-of-memory handler.
tag (input) The memory tag.

Description

This function modifies the location of the out-of-memory error jump buffer. The errloc parameter is the value of a C jump buffer, which you establish by calling setjmp. Control will be passed to that location (via a call to longjmp) when there is not enough memory to satisfy an allocation request (for example, a call to psp_getMemory) on the tag. An errloc value of NULL indicates that NULL will be returned by memory allocation functions when there is insufficient memory.

Required Headers

#include "psp.h"

Libraries

Library Name Description
rdmpsp11 Platform Support Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

None.

See Also

psp_createTag

Example

#include "psp.h"

jmp_buf   memjmp;


...
if (setjmp(memjmp)) {
    /* free any previously allocated memory and return error */
    psp_freeTagMemory(tag, 0);
    return S_NOMEMORY;
}
...
psp_resetTag(memjmp, tag);
