psp_get_pid

Returns the current process ID.

Prototype

uint32_tpsp_get_pid(
    void);

Parameters

None.

Description

This function returns the current processes ID in the form of an unsigned 32 bit integer.

Required Headers

#include "psp.h"

Libraries

Library Name Description
rdmpsp11 Platform Support Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Returns process ID.

Example

#include "psp.h"

...
void check_id(void)
{
    printf("Process ID: %u\n", psp_get_pid()); 
}
