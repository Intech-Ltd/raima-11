psp_fgetc, psp_fgets, psp_fGetLine, psp_fungetc

Read character(s) from a standard I/O stream

Prototype

int32_t psp_fgetc(
    PSP_FILE_STREAM stream);
char *psp_fgets(
    char           *str,
    int32_t         n,
    PSP_FILE_STREAM stream);
char *psp_fGetLine(
    PSP_FILE_STREAM stream,
    char          **str,
    uint32_t       *n);
int32_t psp_fungetc(
    int32_t         c,
    PSP_FILE_STREAM stream);

Parameters

stream (input) The handle to the PSP_FILE_STREAM file opened with psp_fopen or psp_fopen_std.
str (output) A string of UTF-8 characters.
n (input) The allocated length of str.
c (input) A UTF-8 character to put back in the stream buffer so it will be read again.

Description

These functions correspond to the standard C library functions (minus the "psp_" prefix). The additional parameter, stream, must be a handle returned from psp_fopen or psp_fopen_std.

The psp_fGetLine function is the same as psp_fgets except that it will allocate a buffer for the line, however long the line is. The allocated buffer must be freed from tag 0.

Required Headers

#include "psp.h"

Return Codes

A UTF-8 character, or EOF for error. Or a pointer to a string, or NULL for error.
