psp_freeTagMemory

Free all memory associated with a tag

Prototype

void psp_freeTagMemory(
    PSP_MEMTAG tag,
    int16_t    freetag 
);

Parameters

tag (input) The memory tag.
freetag (input) A flag indicating whether to also free the memory tag. To free the tag, specify 1. To allow use of the tag for future memory allocations, specify 0.

Description

This function frees all of the memory that was allocated under a memory tag. It also allows you to specify whether to free the tag or leave it for future memory allocations.

Required Headers

#include "psp.h"

Libraries

Library Name Description
rdmpsp11 Platform Support Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

None.

Example

#include "psp.h"

...
if (setjmp(*memjmp)) {
    /* free any previously allocated memory and return error */
    psp_freeTagMemory(tag, 0);
    return S_NOMEMORY;
}
...
psp_resetTag(memjmp, tag);
