psp_iniInt32

Read a 32-bit integer value from an INI file

Prototype

int32_t psp_iniInt32(
    const char *iniName,
    const char *section,
    const char *tag,
    int32_t    *num);
int32_t psp_iniInt32W(
    const wchar_t *iniName,  /* UNICODE string */
    const wchar_t *section,  /* UNICODE string */
    const wchar_t *tag,      /* UNICODE string */
    int32_t       *num);

Parameters

iniName (input) The name of the INI file to search
section (input) The configuration file section that contains the entry
tag (input) The name of the configuration variable being read
num (output) The 32-bit integer value identified by tag.

Description

An INI file named in iniName will be searched for a section called section and a tag=value pair within the section. This function returns the integer value that matches the tag.

The file is formatted as follows: 

[section]
tag1=value1
tag2=value2
 
; Comments follow semi-colons
