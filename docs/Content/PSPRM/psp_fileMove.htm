psp_fileMove

Move the contents of a file to a second file

Prototype 

int32_t psp_fileMove(
    const char *src,
    const char *dest);
int32_t psp_fileMoveW(
    const wchar_t *src,    /* UNICODE string */
    const wchar_t *dest);  /* UNICODE string */

Parameters

src (input) The name of the source file.
dest (input) The name of the destination file.

Description

This function will attempt to rename a file from src to dest. If that fails, src will be copied to dest and src will be removed. (On VxWorks, the file is always copied). The file paths must be in the local operating system's format. They may be absolute or relative paths.

Required Headers

#include "psp.h"

Libraries

Library Name Description

Return Codes

PSP_OKAY

PSP_FAILED
