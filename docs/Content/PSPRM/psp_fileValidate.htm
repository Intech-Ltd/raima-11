psp_fileValidate

Validate the existence of a file

Prototype

int32_t psp_fileValidate(
    const char *name);
int32_t psp_fileValidateW(
    const wchar_t *name);  /* UNICODE string */

Parameters

name (input) The name of the file to be validated.

Description

This function confirms that a file exists. The file path must be in the local operating system's format. It may be absolute or relative paths.

Required Headers

#include "psp.h"

Libraries

Library Name Description
rdmpsp11 Platform Support Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

1 if the file is found or 0 if the file is not found.
