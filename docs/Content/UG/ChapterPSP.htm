Develop Platform Independent Applications

13.1 Introduction

RDM has been developed to run on several operating systems. For each operating system, an operating system specific module has been developed that presents a uniform API to the rest of the RDM functions. This section publishes several of the functions that may also be used by the application program.

Using the Platform Support Interface

The RDM Platform Support interface provides an efficient interface to such sophisticated operating system tasks as:

  • Concurrency control
  • Dynamic memory allocation
  • File management
  • Stream management
