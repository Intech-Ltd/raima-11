Installing RDM on Windows

This section covers the installation of RDM binary distributions that are provided for Windows in the form of compressed self-extracting installers with the extension .exe. The Engine package must be installed first, then add-on components may be installed.

These instructions are for installing a standard desktop/server version. For installing an embedded/cross compiled platform, see here.

Some registry entries are added for version and install path information only, however no other changes to the environment are made. These entries will be removed upon uninstall.

Engine Installation

This is the base install, and must be installed first.

  • 1. To begin the install, open the installer:

    • NOTE: On Windows Vista and Windows 7, you may be prompted for your password for Administrator level access.

    2. When prompted, accept the license and confirm the install location.

    NOTE: It is highly recommended that the default install location is used, ie C:\Raima\RDM\11.0. It is also not recommended to have more than one install per minor version, ie: one 10.0 and one 10.1, but not multiple of each.

    3. The installer will then install the Engine into the specified directory, and create a shortcut link in the Start menu.

    Silent/Quiet Installation

    To install RDM from the command line, without any graphical windows, simply execute the installer as follows:

    C:\>rdm-engine-bin-win-x32-11.0.0.741.exe /S /D=[install dir]

    Where:

    /S - Silent install

    /D=[install dir] - Specify target install directory

    Both should be used together.

    Uninstall

    Choose uninstall from the RDM Start menu item, or run uninstall.exe from the install folder. Warning: all files in the installation directory will be removed!

    NOTE: Always remove the software using the uninstall option. Do not attempt to remove by deleting the installation directory manually.
