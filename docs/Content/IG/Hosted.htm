Hosted Platform

This section covers the installation of RDM target distributions that are provided for various host-build platforms in the form of compressed self-extracting archives with the extension .run. The Host platform Engine and all desired add-ons must be installed prior to installing & building the target platform.

Host Installation

This is the base install, and must be installed first. Installing from a Bourne compatible shell is recommended.

  • 1. Install the Engine host platform package supported by the target, following the respective Install Guide for Unix or Windows. The following is a list of which host platforms are supported for each target platform:
    •  Wind River VxWorks
      • Linux x32
      • Solaris Sparc32
      • Windows x32
    • Apple iOS
      • Mac OS X x64
    • Green Hills Integrity
      • Linux x32
      • Windows x32
    • QNX Neutrino
      • Linux x32
      • Windows x32

      Note: The iOS target platform is combined with the install for it's host platform, Mac OS X. For installing on the iOS target platform, please see the specific instructions.

    2. Install any desired add on components for the host platform.

    Target Installation

    All Target installs require that the Host Platform components are already installed. Target packages will be in the form:

    rdm-target-src-[HOST]-[TARGET]-[VERSION]

    Note that target platforms consist of only a single package.

    1. Install the package using the command line (Unix host), or graphical installer (Windows host). Choose the directory of the Host Platform install.

    2. The installer will then check that the version and patch level you are attempting to install matches the version and patch level of the existing Engine installation.

    - Installation Complete.

    Compiling

    After installing the host and target packages on the development host, you will then need to cross-compile for the target.

    Note that it is not required to re-compile the host platform, unless debug symbols are needed

    Please follow the specific guide for the target platform:

    • WindRiver VxWorks
    • QNX Neutrino
    • Green Hills Integrity

    Uninstall

    To uninstall, simply remove the installation directory.

    shell> rm -rf /home/build/rdm
