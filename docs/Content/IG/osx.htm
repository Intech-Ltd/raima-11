Installing RDM on Mac OS X

This section covers the installation of RDM binary distributions that are provided for Mac OS X 10.5 and above, as well as RDM Mobile packages for iOS 4.0 and above. For source package installations, please refer to the Unix/Linux Installation Guide.

In this section:

  • Installation
  • Upgrade
  • Uninstall

Installation

The installer is a standard Mac OS installer, with the extension .pkg. All installations will install into the directory /opt/Raima.

  1. To begin, open the installer package file (.pkg), labeled with rdm-workgroup-obj-osx or rdm-workgroup+-obj-osx for RDM Workgroup, and rdm-mobile-ios or rdm-mobile+-ios for RDM Mobile.

  1. Follow the instructions given by the installer, and accept the license agreement.

  1. Select the components you wish to install. At a minimum, the base SDK must be installed.

  1. The completed installation is located at /opt/Raima/RDM/

Next Steps: See how to generate an XCode Project with CMake to build and run the examples.

Upgrade

An existing installation may be upgraded by simply running the new installer, an uninstall is not necessary. Note that this is only valid for new patch versions, where only the patch number is incremented. New major version updates will be installed in addition to the previous version. For example, 10.1.0 will be upgraded to 10.1.1, while 10.2 would be a new installation. The patch upgrade will simply update files in the same installation directory, /opt/Raima/RDM_10.1/, while the major update will create a new directory alongside the existing, in /opt/Raima/RDM_10.2/.

To upgrade an existing version, simply run the installer of the new version, following the same procedure as "Installation" above. Please make sure to read the "Important Information" section in the installer, to see all changes and fixes for the version.

When upgrading, make sure that the Engine installer is run first, followed by all additional components.

Uninstall

To remove RDM from your system, simply run the "Uninstall.pkg" located in the installation directory.

WARNING: This will remove all contents of the install directory at /opt/Raima/RDM_11.0/. If you have any data you wish to save, please move or copy it out of the installation PRIOR to running the uninstall. Note that if you have other versions of RDM installed alongside, they will be left untouched.
