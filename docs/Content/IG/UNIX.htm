Installing RDM on Unix / Linux

This section covers the installation of RDM binary distributions that are provided for various Unix platforms in the form of compressed self-extracting archives with the extension .run. The Engine package must be installed first, then add-on components may be installed.

These instructions are for installing a standard desktop/server version. For installing an embedded/cross compiled platform, see here.

Engine Installation

This is the base install, and must be installed first. Installing from a Bourne compatible shell is recommended.

  • 1. To begin the install, execute the command from a terminal as follows:
    • shell> ./rdm-engine-bin-OS-VERSION.run

    NOTE: Ensure that the command is run from a user account that has the appropriate permissions for the desired installation directory.

    2. When prompted, enter the desired installation directory (either absolute path, or relative to the current working directory):

    - Enter directory for new install path:
/home/build/rdm

    NOTE: The installer will create the directory RDM_11.0/ within the directory specified.

    3. The installer will then install the Engine.

    - Installation Complete.

    Other Components (Distributed, High Availability, and Data Flow)

    All component installs require that the Engine is already installed.

  • 1. To begin the install, execute the command from a terminal as follows:

    • shell> ./rdm-engine-dist-OS-VERSION.run

    NOTE: Ensure that the command is run from a user account that has the appropriate permissions for the desired installation directory.

    2. When prompted, enter the directory of the existing Engine install.

    - Enter the existing RDM install location:
/home/build/rdm

    3. The installer will then check that the version and patch level you are attempting to install matches the version and patch level of the existing Engine installation.

    - Installation Complete.

    Uninstall

    To uninstall, simply remove the installation directory.

    shell> rm -rf /home/build/rdm
