Package Names

RDM package distributions have names of the form:

rdm-PKG-bin-OS-ARCH-VERSION.PATCH.BUILD.EXT

where:

PKG

Indicates the type of package:

OS

Indicates operating system and hardware architecture:

Platform
aix AIX
fbsd FreeBSD
hpux HPUX
int Integrity
ios iOS
lnx Linux
neu Neutrino
osx64 Mac OS X
sol Solaris
vxwks VxWorks
win Windows
wince Windows CE / Windows Mobile
Architecture
x32 Intel 32bit
x64 Intel 64bit
ia32 Itanium 32bit
ia64 Itanium 64bit
ppc32 PowerPC 32bit
ppc64 PowerPC 64bit
risc32 PA Risc 32bit
risc64 PA Risc 64bit
arm ARM Cortex Series (ARMv7 instruction set)

VERSION

Indicates product version and patch level (for example 11.0.0).

BUILD

The build number

EXT

The filename extension, run on Unix, exe on Windows.
