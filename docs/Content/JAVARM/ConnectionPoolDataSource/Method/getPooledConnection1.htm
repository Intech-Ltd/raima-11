getPooledConnection(String user, String password)

Syntax:

PooledConnection getPooledConnection(String user,
                                     String password)
                                     throws SQLException

Description:

Attempts to establish a physical database connection that can be used as a pooled connection.

Raima Database Manager does not support usernames or passwords, so these parameters are silently ignored.

Parameters:

user

the database user on whose behalf the connection is being made

password

the user's password

Returns:

a PooledConnection object that is a physical connection to the database that this ConnectionPoolDataSource object represents

Throws:

SQLException - if a database access error occurs

Reference:
