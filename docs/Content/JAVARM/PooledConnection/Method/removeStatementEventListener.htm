removeStatementEventListener(StatementEventListener listener)

Syntax:

void removeStatementEventListener(StatementEventListener listener)

Description:

Removes the specified StatementEventListener from the list of components that will be notified when the driver detects that a PreparedStatement has been closed or is invalid.

Parameters:

listener

the component which implements the StatementEventListener interface that was previously registered with this PooledConnection object

Reference:

For more information, reference JDBC documentation for: PooledConnection.removeStatementEventListener(StatementEventListener)
