rsqlDropDB

Drop (delete) a database

Prototype

RSQL_ERRCODE EXTERNAL_FCN rsqlDropDB(
 	HCONN          hdbc,
 	char          *dbspec)

Arguments

hConn (input) Connection handle.
dbspec (input) Pointer to string that identifies the name of the database to be dropped.

Description

Function rsqDropDB can be called to drop the database identified by the dbspec string as formed based on the following syntax:

dbspec: dbname[HostComputerName[:ddddd]]

where HostComputerName is just that and ddddd is the five digit TCP/IP port number on which that TFS is listening (default is 21553).

If the database is open then you only need to specify the dbname and the call to rsqlDropDB will close it. The database is closed even when the function returns an error (except for errTRACTIVE).

Status errNODB is returned if the database cannot be found. Status errDBINUSE is returned if another task or user has the database open. Status errTFSFAILURE is returned when a connection to the specified TFS cannot be made.

A successful call to rsqlDropDB completely deletes the database and is irrecoverable (i.e., a call to rsqlTransRollback or rollback statement cannot undo the call to rsqlDropDB).

Required Headers

#include "rsql.h"

Example

#include "rsql.h"
...
HCONN hdbc;
RSQL_ERRCODE stat;
	...
	/* drop the bookshop database */
	rsqlAllocConn(&hdbc, 0);
	rsqlDropDB(hdbc, "bookshop");
	...
