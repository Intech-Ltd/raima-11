RdmDataAdapter class

Description:

The RdmDataAdapter class, derived from DbDataAdapter, is a class that servers as a bridge between a DataTable or DataSet and the data in an RDM database. Data is moved from the database to the DataTable or DataSet when a Fill method is called and changes are made back to the database when an Update method is called.
