Objective-C/Cocoa API

The Objective-C 2.0 interface to RDM is designed to augment the RDM navigation on Mac OS X and iOS platforms by creating custom objects to represent records and interface with the database. This combines the performance of RDM with the intuitiveness of an object oriented interface, making it easy to integrate with Cocoa applications.

The API is fully compatible with both Mac OS X 10.7 Lion and iOS 5, including support for Automatic Reference Counting (ARC).

 

Documentation

Documentation is available in html format, and as an XCode Documentation Feed. It is highly recommended to subscribe to the XCode feed, as updates to the documentation will be pushed automatically. See the instructions below to install the feed.

 

XCode Documentation Feed

Setting up the documentation feed requires two steps, first subscribing to the feed, then downloading the actual documentation. Once the feed is subscribed to, new updates will be pushed automatically. When installed, the documentation may be viewed directly from XCode, even without an Internet connection.

 

 

Subscribing to the Feed and Installing the Documentation Set

1. From within XCode, choose XCode > Preferences, then Documentation:

2. Click on the '+' to add a new entry:

3. The feed dialog will appear. Enter "http://docs.raima.com/rdme/feeds/com.raima.rdme.11_0.objective-c.atom", and click Add.

Installing the Documentation

In the Preferences > Documentation window described above, click Get to download the docset.

5. You will be prompted for your password to allow installation

Viewing the Documentation

To view the documentation, choose Organizer in the toolbar

...then Documentation

Congratulations, you now have the documentation subscribed to and installed in XCode! When updates are available, they will be downloaded automatically.
