d_keybuild

Rebuild all keys for the database

Prototype

int32_t d_keybuild(
    DB_TASK *task, 
    int32_t  dbn);
int32_t d_keybuildEx(
    const KEYBUILD_PARAMS *params,
    DB_TASK               *task,
    int32_t                dbn);
int32_t d_keybuildExW(
    const KEYBUILD_PARAMSW *params,
    DB_TASK                *task,
    int32_t                 dbn);

Parameters

params (input) A pointer to a KEYBUILD_PARAMS structure.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

Function d_keybuild recreates all the key files for the given database. Function d_keybuildEx can be used to build individual key files. See keybuild for a complete description.

KEYBUILD_PARAMS structure
Member Type Description
quiet DB_BOOLEAN No output except for errors.
report_frequency uint16_t The -f option. Report progress to stdout every N records. Set to 0 for no report.
keyfile_name const char * Specify a single key file to build. Only this key file will be built.
stdout_file const char * This specifies the name of the file to which output data will be written. If set to PSP_STDOUT, stdout is used. If set to NULL or to an empty string, no output will be generated.

Locking Requirements

This utility requires exclusive access to the database.

Required Headers

#include "dutilproto.h"

Libraries

Library Name Description
rdmutil11 RDM Utility Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-32 S_EXCLUSIVE exclusive access required
-905 S_NOFILE error opening data or key file

See Also

keybuild utility

Example

int32_t          status = S_OKAY;
    DB_TASK         *task;
    KEYBUILD_PARAMS  params;
    char            *keyfile = "tims.k01";

    memset(&params, 0, sizeof(KEYBUILD_PARAMS));

    /* just set the non-zero parameters */
    params.keyfile_name = keyfile;
    params.stdout_file = PSP_STDOUT;

    if ((status = d_opentask(&task)) == S_OKAY) {
        if ((status = d_open("tims", "x", task)) == S_OKAY) {
            status = d_keybuildEx(&params, task, CURR_DB);
            d_close(task);
        }
        d_closetask(task);
    }
