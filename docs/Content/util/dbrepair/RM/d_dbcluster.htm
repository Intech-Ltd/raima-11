d_dbcluster

Database set optimization utility API.

Prototype

int32_t d_dbcluster(
    const DBCLUSTER_PARAMS *params,
    DB_TASK                *task, 
    int32_t                 dbn);

int32_t d_dbclusterW(
    const DBCLUSTER_PARAMSW *params,  /* UNICODE strings in structure */
    DB_TASK                 *task, 
    int32_t                  dbn);

Parameters

params (input) A pointer to a DBCLUSTER_PARAMS structure.
task (input) A pointer to a DB_TASK structure.
dbn (input) The database number.

Description

This function invokes the dbcluster utility from your application. See the dbcluster utility page for a description of its operation.

DBCLUSTER_PARAMS structure
Member Type Description
quiet DB_BOOLEAN No output except for errors.
verbose DB_BOOLEAN Show detailed output of the operations performed on the database
nBlock uint16_t Size of an internal iteration block. This is the number of sets to process per iteration. This should be set to 0 to use the default.
numSets uint16_t The number of set names listed in the sets parameter value. This should be 0 if you want to cluster all sets in the database
sets const char ** An array of set names listing the specific sets that will be clustered. If this list is empty all sets in the database will be sorted.
stdout_file const char * This specifies the name of the file to which output data will be written. If set to PSP_STDOUT, stdout is used. If set to NULL or to an empty string, no output will be generated.

Currency Changes

Currency is invalidated by using this API.

Locking Requirements

This utility requires exclusive access to the database.

Required Headers

#include "dutilproto.h"

Libraries

Library Name Description
rdmutil11 RDM Utility Library

See Library Naming Conventions section for full library name and a list of library dependencies.

Return Codes

Value Name Description
0 S_OKAY normal return, okay
-1 S_DBOPEN database not opened
-2 S_INVSET invalid set
-24 S_NOTLOCKED attempt to access unlocked set/record
-32 S_EXCLUSIVE exclusive access required
-904 S_NOMEMORY out of memory
-939 S_READONLY unable to update file due to READONLY option

See Also

dbcluster utility

Example

int32_t MAIN(int32_t argc, const char *const *argv)
{
    const char       *dbname = "tims";
    int32_t           status = S_OKAY;
    DB_TASK          *task;
    DBCLUSTER_PARAMS  params;

    memset(&params, 0, sizeof(DBCLUSTER_PARAMS));	
    params.stdout_file = PSP_STDOUT;

    if ((status = d_opentask(&task)) == S_OKAY) {
        if ((status = d_open(dbname, "x", task)) == S_OKAY) {
            status = d_dbcluster(&params, task, CURR_DB);
            d_close(task);
        }
        d_closetask(task);
    }
    return(0);
}
