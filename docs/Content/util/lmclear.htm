lmclear

Clears lock manager session (all platforms)

Prototype  

lmclear [-h] [-B] [-V] [-q|[-stdout file]] [-stderr file] -s|-u username
        [-a tfservername]

Description  

The lmclear utility has two main uses.

  • Clear a user with whom communication has been lost, and this is unknown to the lock manager. (Clearing a user means to free any locks the user has been granted.
  • Display a status report on the lock manager.

Options  

-h Display this usage information
-B Do not display the banner
-V Display the version information
-q Quiet mode. No information will be displayed
-stdout file Redirect stdout to the specified file
-stderr file Redirect stderr to the specified file
-s Prints a status report for all users.
-u userid Instructs the lock manager to clear the user specified by userid. This option is not necessary if the environment variable DBUSERID specifies the user.
-a Specifies the tfserver name
