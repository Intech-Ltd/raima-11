insstr

Insert a string into another string

Syntax

insstr(strexp1, num1, num2 ,strexp2)

Parameters

strexp1 The string to be inserted into.
num1 The character offset from the beginning of strexp1 where the insertion will take place. The first position is 1, not 0.
num2 The number of characters in strexp1 to be replaced.
strexp2 The string to use as the replacement.

Description

This scalar string function retrieves the string that results from replacing num2 number of characters in the string that results from the evaluation of strexp1, beginning at position num1. The insstr function replaces these characters with the string that results from evaluation of strexp2.

Example

Function                           Result
insert("Wayne", 2, 1, "Bill")	   "WBillyne"
