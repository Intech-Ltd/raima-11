SQL Virtual Table Function Reference

Function Description
vtFetch Fetch the next row in the virtual table
vtInsert Process execution of an insert statement into a virtual table
vtRowCount Return estimate of number of rows in virtual table
vtSelectClose Close select statement execution access to virtual table
vtSelectCount Return actual number of rows in virtual table
vtSelectOpen Process execution of SQL statement access to virtual table
